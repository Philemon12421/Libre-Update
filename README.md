# Libre - Digital Library & Explorer 📚

Libre is a sleek, mobile-first application designed to bring order to your digital life. Built with a focus on simplicity, privacy, and community, Libre allows you to manage your local files and discover new books across the web in one unified experience.

Designed by **Drenchack Tech Company** to solve the chaos of scattered files.

## ✨ Features

- 📁 **Local File Management**: Upload and organize PDFs, DOCX, PPT, and Images directly on your device.
- 📄 **Pages (Folders)**: Create custom "Pages" to categorize your collections.
- 🔍 **Universal Book Search**: Search across 8+ major digital libraries (Google Books, Open Library, etc.) without needing any API keys.
- 👁️ **Instant Preview**: Read and preview your files and book discoveries inside the app.
- 🌙 **Modern UI**: A clean white and blue aesthetic with smooth `framer-motion` (motion) transitions.
- 🔐 **Privacy First**: All your library files are stored locally using IndexedDB (Dexie).
- 🌐 **Open Source**: Built for the community, by the community.

## 🛠️ Technology Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (framer-motion)
- **Local Storage**: Dexie (IndexedDB)
- **Icons**: Lucide React

## 📱 Mobile Publication

This project is structured to be compatible with **Expo**. To transition to a native mobile app:
1. Initialize an Expo project.
2. Port the components to React Native (using native-friendly Tailwind variants).
3. Utilize `expo-file-system` and `expo-notifications` for deep mobile integration.
